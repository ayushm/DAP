Generate and Print Reports

 

 

1)      From the HOME screen, click on REPORTS.

 

 

2)      In the top left corner of the REPORTS window, from option panel, select Catalog Items to display a list of library Items.

a.      Select ‘All’ to display a report of all the items in the library. OR

b.      Select the desired Item Type from the drop down list to display a report of all the items of that type in the library.

 

 

3)      In the top left corner of the REPORTS window, from option panel, select Checked Out Items to display a list of library Items checked out to patrons.

a.      Select ‘Sort By Due Date’ to display the report sorted by due-date.

b.      Select ‘Sort By Patron’ to display the report sorted by patron. The report lists the items checked out by patron, due date, overdue indicator (*)  in front of the items that are overdue.

 

4)      You can scroll through the report in the report text area on the screen.

 

 

5)      Once satisfied with the output, click PRINT button with printer icon to print the current report.

a.      Select the appropriate printer settings in the printer dialog box.
