By : Ayush Mehra (Homestead High School FBLA)

 

Program General Information

-          The Library Management System is a useful application that provides intuitive operations to maintain items in a lending library.

-          The application was coded using Java on Windows OS (also compatible on Mac and Linux operating systems)

-          All images used in this application are public domain with appropriate GPL licensing

-          For more information click here

Program Execution instructions

1)      Please copy all the contents of the CD to the desired location.

2)      Double Click on DAP(.jar) to run the library Management System

3)      –OR- 

a.      To run the project from the command line, navigate to the folder where all the files were copied and type the following:

b.      java -jar DAP.jar

4)      Program source code can be found in ‘src’ folder.

5)      Technical Documentation for the Program (Java Docs) can be found here or by double clicking ‘Index.html’ in the ‘javadoc’ folder.

6)      To get more information on using the program click on the following links (Also accessible throughout the program by pressing F1) :
